# Setup Docs
```bash
#install dependencies
npm install

#setup variables.env
    NODE_ENV=<string>
    DATABASE=mongodb://<username>:<password>@<url>.com/<database-name>
    PORT=<dev-env-port>
    SECRET=<string>
    KEY=<string>

#start dev enviorment
npm run start
```
